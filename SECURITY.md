# Security Policy

## Scope

`mnox-ai` is a collection of Claude Code skills — Markdown instructions plus a
handful of helper Python scripts. It ships no network service, no runtime
server, and no credential handling. The realistic security surface is limited
to the local helper scripts under `skills/*/scripts/`.

## Reporting a vulnerability

Please **do not** open a public issue for a security report.

Use GitHub's private vulnerability reporting instead:
**Security → Report a vulnerability** on this repository
(<https://github.com/mnox/mnox-ai/security/advisories/new>).

You can expect an initial response within a few days. This is a personal,
best-effort project, so please be patient with any follow-up.

## Supported versions

Only the latest released version receives fixes.
